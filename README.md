## 📌 Sobre o projeto
A **Locatech API** é uma aplicação backend desenvolvida para estudo de construção de APIs REST com Java e Spring Boot.
## ✅ Funcionalidades implementadas
- CRUD de veículos
- Organização em camadas
- Banco H2 em memória
## 🧱 Stack utilizada
- **Java 21**, **Spring Boot 4.0.5**, **Spring JDBC**, **Spring Web MVC**, **H2**, **Lombok**
## 🏛️ Arquitetura
A aplicação utiliza uma arquitetura em camadas (controllers, entities, repository, service).
